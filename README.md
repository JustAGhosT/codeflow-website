# CodeFlow Website

Marketing website for CodeFlow AI built with Next.js.

## Overview

This is the marketing website for CodeFlow AI, providing information about the platform, features, and getting started guides.

## Technology Stack

- **Next.js**: React framework
- **TypeScript**: Type-safe JavaScript
- **Tailwind CSS**: Utility-first CSS framework

## Development

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

The website is deployed to Azure Static Web Apps. See [`codeflow-infrastructure`](../codeflow-infrastructure) for deployment infrastructure.
